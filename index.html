University of York

Part 1 - A Molecular Replacement Example
Part 2 - Building From Scratch

This workshop is split into two parts. The first is an easy molecular replacement example that starts from the beginning and covers how to drive Coot, fix an existing protein model, and add ligands and waters. The second part assumes you know everything in the first part and covers building a model from an empty map using baton building and NCS.

Many thanks to Paul Emsley for writing Coot and providing tutorial material. For lots more information visit the website (especially The Coot User Manual).
