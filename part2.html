7. Building a protein from scratch

7.1. Skeleton map

A much smaller molecule will be used to practice building a protein from just a map. Save our changes to the current structure using File / Save Coordinates then delete all molecules and maps. Click Extensions / Load tutorial model and data to load the smaller structure, which has two copies of RNase.

Start by going to the C-alpha of residue 2 in the A chain. Make a note of the position of the C-alpha in residue 1 then hide the molecule in the display manager. The backbone trace through the density can be more easily visualised by using a skeleton map Use skeleton map to help visualise where the C-alphas are.

7.2. Baton building

Start from a position of a C-alpha one away from the end. Then you will see an arrow 3.8A in length to place the first C-alpha at the end of the chain. From this first C-alpha you will place a baton back at the place you came from. Then place batons to the other end of the peptide. Click dismiss when done. Convert to a C-alpha trace. Will get a backwards and forwards trace. One should fit better than the other. Real space refine zone.
