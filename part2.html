1. Introduction

This tutorial is a slightly modified copy of this one from the Coot website. It assumes you are familiar with the concepts covered in part 1 of the workshop.

When automatic building fails, typically because the resolution limit of your data is too low, then building the molecule "by hand" may be the only way to proceed. Recognizing the shape of main-chain and side-chain densities is valuable and this tutorial aims to introduce these to you. Note that this tutorial map is an easy map to build into, the side chains are (mostly) clear. Using just a map and a sequence, we will attempt to generate a model. This model can then be validated and refined with Refmac for several rounds. With some experience you should be able to get an R-factor of less than 20% in less than 30 minutes.

2. Starting

Open Coot, then click Extensions / Load tutorial model and data to load the structure. It has two chains with 96 residues in each. Start by going to the C-alpha of residue 2 in the A chain. The idea is to build the complete A chain in the forward direction and compare it to the real structure afterwards.

Make a note of the position of the C-alpha in residue 1 then hide the molecule in the Display Manager until you have finished building, validating and refining. It is also a good idea to hide the difference map because we are not interested in the previously built model.

The backbone trace through the density can be more easily visualised by using a skeleton map. Go to Calculate / Map Skeleton and change it to On.

3. Baton Building

We will now start to “baton build” the map. Go to Calculate / Other Modelling Tools to open another toolbar and click C-alpha Baton Mode. A set of controls will appear, as well as a white baton and a set of pink points that show possible places for a C-alpha that are 3.8 Å away from the current position.

Remember, when you start, you are placing a CA at the baton tip for residue 1 (the N-terminus). After placing CA for residue 1, you will get a choice of positions for residue 2, which is currently at the centre of the view. This might seem that you are "double-backing" on yourself - which can be confusing the first time.

Press Try Another and Previous Tip Position to cycle through the points, Lengthen and Shorten to change the baton length, and Accept to place a CA and move onto the next residue. If none of the probe points are suitable you can use b to toggle baton swivel mode. If you have installed the template keybindings (through Extensions / Settings) then you can use the shortcut A (Shifta) to accept the baton position.

Build from the N-terminus to the C-terminus. There are 96 residues to build and it takes about 15 minutes or so. If you make a mistake you can press Undo to move back one residue. The last residue should be a cysteine that forms a disulphide bond. Click Dismiss on the baton controls when done. There may also be a molecule called Baton Atom Guide Points that needs deleting. You can probably also turn the map skeleton off at this stage.

4. Convert to Mainchain

Now we need to turn these CA positions into mainchain. Use Go To Atom to centre on the first residue of the Baton Atoms molecule. Then use Ca Zone -> Mainchain in Other Modelling Tools and click on the first CA. You will need to wait several seconds while it builds (note that you need at least 6 baton points for this to work). Two new molecules will be created: mainchain-forwards and mainchain-backwards traced in different directions. We know the forward direction is correct (see how much better the carbonyls fit) so the other molecule can be deleted.

The backbone now needs to be tidied up. Go to Extensions / All Molecule / Stepped Refine, pick mainchain-forwards and watch it as it goes. Is it making mistakes? That refinement may have gone to quickly to make a note of problem areas. Use Density fit analysis and real space refinement to correct anything that's wrong.
